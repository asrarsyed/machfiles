#### IosevkaTerm Nerd Font

This version of `iosevka` font is used in terminal to fallback on some symbols, such as icons, which are not included in `iosevka-custom`

You can download it from [nerdfonts.com/font-downloads](https://www.nerdfonts.com/font-downloads)

The direct link: https://github.com/ryanoasis/nerd-fonts/releases/download/v3.1.1/IosevkaTerm.zip
